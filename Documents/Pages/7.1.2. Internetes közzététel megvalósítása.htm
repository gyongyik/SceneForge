7.1.2. Internetes közzététel megvalósítása

A SceneForge közzététele a GitHub segítségével történt. A GitHub egy olyan interneten keresztül elérhető szolgáltatás, ami lehetővé teszi azt, hogy a program fejlesztésén egyszerre többen is dolgozhassanak. A feltöltött programok kódjáról bárki készíthet magának egy másolatot, amin tetszőleges változtatásokat hajthat végre. A program különböző változatai a változtatásokat követően összefésülhetők egymással. Weblap: https://github.com
