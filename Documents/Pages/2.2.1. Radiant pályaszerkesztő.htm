2.2.1. Radiant pályaszerkesztő

A Radiant az id Software által kifejlesztett pályaszerkesztő program, amiből több változat létezik. Legismertebb a GtkRadiant, ami a Quake III (id Tech 3) és a Doom 3 (id Tech 4) játékmotorral is használható. Az alábbi táblázat három népszerű Radiant változat internetes elérhetőségét tartalmazza.

Radiant változatok összehasonlítása
GtkRadiant http://icculus.org/gtkradiant id Software által fejlesztett változat
NetRadiant http://icculus.org/netradiant/files id Tech 3 játékmotorra optimalizált
DarkRadiant  http://www.darkradiant.net  id Tech 4 játékmotorra optimalizált 

A különböző Radiant változatok közül jelenleg a DarkRadiant a legfejlettebb. A többi változathoz képest olyan hasznos újításokkal rendelkezik, mint például a Python szkriptek alkalmazása, vagy a fények valós időben történő kiszámítása és megjelenítése.
